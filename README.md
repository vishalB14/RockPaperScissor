# 🎮 Rock Paper Scissors Game (Python)

A simple Rock Paper Scissors game built using Python.  
This project allows a user to play against the computer in a command-line interface.

---

## 🚀 Features

- 🎮 Play Rock, Paper, Scissors against the computer
- 🤖 Random computer move generation
- 🏆 Win / Lose / Draw result logic
- 🔁 Multiple rounds support
- 🧾 Clean and simple Python implementation

---

## 🛠️ Technologies Used

- Python 3
- Random module

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash

cd RockPaperScissor
```

### 2️⃣ Run the Game

```bash
python StonePaperScissor.py
```

---

## 🎯 How to Play

1. Run the Python file.
2. Enter your choice:
   - rock
   - paper
   - scissors
3. The computer randomly selects its move.
4. The winner is displayed.

---

## 🧠 Game Logic

- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock
- Same choice = Draw
